# Cinematic Portfolio

An ultra-premium, cinematic digital experience designed for high-end developers and digital architects. This portfolio focuses on immersive storytelling through motion, depth, and interactivity.

## ✨ Key Features

- **Cinematic Preloader**: A 3-stage visual introduction that sets the tone for the experience.
- **WebGL Background**: High-performance galaxy-style particle field powered by Three.js.
- **GSAP Animations**: Precision-engineered motion and scroll-triggered reveals.
- **Custom Cursor System**: A multi-layered, interactive cursor that responds to user intent.
- **Kinetic Typography**: Dynamic text animations that breathe life into the content.
- **Magnetic Interactions**: UI elements that pull the cursor in, creating a tactile feel.

## 🛠️ Technology Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Logic**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (Premium Design System)
- **Animation**: [GSAP](https://gsap.com/) & [ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/)
- **3D/WebGL**: [Three.js](https://threejs.org/) & [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber)
- **Smooth Scroll**: [Lenis](https://github.com/darkroomengineering/lenis)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/s4meer-dev/Portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) to view the experience.
